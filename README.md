            - ...

- 本地搭建流程
    1. 安装 **[node.js](http://nodejs.org/)**
    - 全局安装gulp:
        - **npm install -g gulp**
    - 全局安装bower
        - **npm install -g bower**
    - 项目根目录下安装依赖包
        - **npm install**
    - 构建命令：
        - **生产环境**: **gulp prod**
        - **开发环境**: **gulp dev**
